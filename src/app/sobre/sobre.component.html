Angular::Notícias

O Angular::Notícias é um software experimental desenvolvido com base no Framework Angular e utilizado como projeto de desenvolvimento da disciplina Linguagem de Programação para a Web.

Recursos

Os principais recursos (ou funcionalidades) do software são:

  • Tela "home" apresenta a notícia de destaque e as três notícias mais recentes
  • Tela "noticia" apresenta a lista de todas as notícias, ordenadas pela mais recente
  • Tela de leitura da notícia permite ler o conteúdo de uma notícia
  • Utilização de Bootstrap para a interface gráfica
  • Utilização do Font-awesome para ícones
  • Arquitetura do software baseada em serviços que consultam os dados das notícias
  • Uso de rotas para organização das telas em URLs
